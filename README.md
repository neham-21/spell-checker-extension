Spell Checker Chrome Extension

A Chrome Extension that detects misspelled words and suggests the closest correct words using the Levenshtein Distance Algorithm. This project demonstrates the practical implementation of Design and Analysis of Algorithms (DAA) concepts in a real-world application.

Features:
1.Detects spelling mistakes
2.Suggests the closest matching word
3.User-friendly Chrome extension interface
4.Uses Dynamic Programming for edit distance calculation
5.Fast and lightweight implementation


Algorithms Used:
LEVENSHTEIN DISTANCE

The extension calculates the minimum number of operations required to transform one word into another.

Operations allowed:

Insertion
Deletion
Replacement

Time Complexity:

O(mn)

Where:

m = length of first word
n = length of second word

Technologies Used:
HTML
CSS
JavaScript
Chrome Extension APIs
